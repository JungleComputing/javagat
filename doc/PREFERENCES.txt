Generic preferences:

preference key                              possible value(s)         default value           meaning
-----------------------------------------------------------------------------------------------------
"adaptors.local"                            "true", "false"           false                   Use only local adaptors.
"killJobsOnExit"                            "true", "false"           true                    Kill submitted jobs when exiting.
"XXX.adaptor.name"                          adaptor name              n.a.                    Use adaptor with the specified name for type XXX.
                                                                                              For example, you can set "ResourceBroker.adaptor.name" to "globus" to force
                                                                                              the use of the globus adaptor. The adaptor name is the class name of 
                                                                                              the adaptor, minus the type and the word "Adaptor".
                                                                                              So, for the GlobusResourceAdaptor class, the adaptor name will be "Globus".
                                                                                              For the SftpFileInputStreamAdaptor class, the name is "Sftp"
                                                                                              Names are not case sensitive, so you can also use "globus" and "sftp" or "SFTP".

globus generic
"globusCert",                                                                                 A string that contains the globus certificate in UTF-8 format. 
                                                                                              You can use this if your application already initialized your credential 
                                                                                              (normally not needed).
