This package contains classes and interfaces which are used to provide input and output employing streams and the file system. The most important classes and interfaces are File, LogicalFile, Streamable, Stream, FileStream, and the corresponding capability provider interfaces. An instance of the class File is an abstract representation of a physical file and is used to preform operations on an entire file. An instance of the class LogicalFile is an abstract representation of a set of identical physical files and is used to facilitate data transfer. The Streamable interface specifies the methods that a class representing a connection to an external entity must implement. An instance of the class Stream represents an open connection to another process and is used to send data to other processes. An instance of the class FileStream represents an open file and is used to read data from and write data to an open file.The various capability provider interfaces are used by an adaptor writer to implement the functionality contained in this package.
